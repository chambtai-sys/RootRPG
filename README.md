# 🎮 RootRPG

**A complete, cyberpunk-inspired RPG engine built entirely in Windows Batch.**

![Tech: Windows Batch](https://img.shields.io/badge/Tech-Windows_Batch-4D4D4D?style=for-the-badge&logo=windows&logoColor=white)
![Style: ASCII](https://img.shields.io/badge/Style-ASCII_Art-00FF00?style=for-the-badge)
![License: MIT](https://img.shields.io/badge/License-MIT-green.svg?style=for-the-badge)

RootRPG is a minimalist terminal adventure that takes you deep into the 'Under-Grid'. Manage your resources, upgrade your system, and fight security sentinels in a quest to save the digital world.

## ✨ Features

- **🛡️ Combat System**: Tactical turn-based combat with attack, defend, and high-risk hacking options.
- **💰 Data-Shop**: Exchange earned credits for repair kits and overclock modules.
- **🌐 Procedural Elements**: Random damage calculation and hacking success rates.
- **📟 Immersive UI**: Complete ASCII-art-driven interface with a retro green matrix theme.
- **💾 Lightweight**: Zero dependencies. Runs on any Windows machine via the native command prompt.

## 🛠️ Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/chambtai-sys/RootRPG.git
   cd RootRPG
   ```

2. **Launch the Game**:
   Simply double-click `game.bat` or run it from CMD/PowerShell:
   ```cmd
   game.bat
   ```

## 🎮 Gameplay Guide

| Action | Description |
|---|---|
| **Mission** | Enter Sector 7 to engage in combat and retrieve data. |
| **Inventory** | View your current upgrades and modules. |
| **Shop** | Spend credits to boost your performance. |
| **Hack** | High-risk, high-reward move in combat that can end a fight instantly. |
